Replaces the term description and user bio textareas with a TinyMCE editor.

== Installation ==
1. Upload `slimline-tinymce` folder to the `/wp-content/plugins/` directory
2. Activate the `Slimline TinyMCE` through the \'Plugins\' menu in WordPress
3. Term Description and User Bio textareas will automatically be replaced with a TinyMCE editor.
